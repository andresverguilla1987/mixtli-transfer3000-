# Mixtli Transfer – Combo (Frontend + Backend)
## 1) Backend (Render)
- `cd backend`
- `npm install`
- Variables de entorno según `.env.example`
- Start: `node --enable-source-maps server.js`

## 2) Frontend (Netlify)
- `cd frontend`
- Edita `_redirects` con tu URL de Render
- Deploy en Netlify (Publish dir = `frontend/`)
