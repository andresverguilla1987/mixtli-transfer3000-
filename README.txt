Deploy: Render (Node 22+)
-------------------------
1) Create a Web Service (Node).
2) Set Build Command: `npm install --no-audit --no-fund`
3) Set Start Command: `node --enable-source-maps server.js`
4) Add the environment variables from `.env.example`.
5) Ensure `ALLOWED_ORIGINS` includes your Netlify domain.
6) Ensure `BACKEND_PUBLIC_ORIGIN` is the Render URL of this service.
